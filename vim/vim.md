## move cursor

```
H M L
W B

%

{ }
[[ ]]

gD 		// jump to definition.
''		// back to last cursor
```

## visual

```
viw			// visual select word.
ciw 		// cut word than insert.
yiw			// just copy select word.
```

## bookmark

```
m[char]		// set book mark [char].
`[char]		// jump to book mark [char].
```

## others

```
> 			// indent
<			// indent
```

## config

```
colorscheme default
set encoding=utf-8
set mouse=
```

## command

```
e open		// open file.
```
